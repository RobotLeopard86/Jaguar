class ScopedReadView

Abstraction for accessing a scoped area of the byte stream.

This class may not be copied or moved; it belongs to the Reader that created it. As such, it is prone to being messed up by other stream operations. You should avoid performing other stream operations until you are done with this object.

inline void Read(R &out, uint32_t byteCount)

Parameters:

  • out – The destination to write to

  • byteCount – The number of bytes to read

Throws:

  • std::runtime_error – If the byte count to read exceeds the number of remaining bytes

  • std::runtime_error – If an IO error occurs while reading
