Enum Documentation

enum class libjaguar::TypeTag : uint8_t

All valid type specifiers in a Jaguar stream.

Values:

enumerator String

UTF-8 string.

enumerator ByteBuffer

Raw bytes blob.

enumerator Substream

Embedded independent Jaguar stream.

enumerator Boolean

True/false (bool)

enumerator Float32

Single-precision (32-bit) IEEE 754 LE floating-point number (float)

enumerator Float64

Double-precision (64-bit) IEEE 754 LE floating-point number (double)

enumerator SInt8

8-bit signed integer (int8_t)

enumerator SInt16

16-bit signed integer (int16_t)

enumerator SInt32

32-bit signed integer (int32_t)

enumerator SInt64

64-bit signed integer (int64_t)

enumerator UInt8

8-bit unsigned integer (uint8_t)

enumerator UInt16

16-bit unsigned integer (uint16_t)

enumerator UInt32

32-bit unsigned integer (uint32_t)

enumerator UInt64

64-bit unsigned integer (uint64_t)

enumerator List

List of other values.

enumerator UnstructuredObj

Object with no predefined layout, dictionary.

enumerator StructuredObj

Object with predefined layout.

enumerator StructuredObjTypeDecl

Declaration of an object type layout.

enumerator ScopeBoundary

End of object scope marker.

enumerator Vector

2, 3, or 4-component vector of numbers

enumerator Matrix

Matrix of numbers, size from 2x2 to 4x4.
