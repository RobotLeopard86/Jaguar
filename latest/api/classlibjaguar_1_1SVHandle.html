class SVHandle

Safe, move-only, non-owning wrapper for accessing a ScopedView.

Public Functions

inline bool IsHandleValid()

Check if the handle is valid.

Warning

Even if this method returns true, the view itself may not still be valid (it may be exhausted). Always check the view’s IsValid method as well to determine if the view is safe to use.

Returns:

If the stored view is still safe to access

inline ScopedView *operator->()

Access the view.

Returns:

The contained ScopedView object — do not delete this pointer!
